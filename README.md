# Talisma

AI-powered recruitment ecosystem for modern hiring teams.

## Overview

Talisma is an intelligent Applicant Tracking System (ATS) that combines AI-driven candidate matching, resume parsing, interview generation, and hiring analytics into a unified platform. It helps recruiters streamline sourcing, evaluation, and hiring workflows.

## Features

### 📊 Hiring Command Center
- Real-time hiring KPIs
- Conversion funnel and pipeline health metrics
- Sourcing channel analytics

### 💼 Requisition Management
- Create and manage roles with skills, salary, and seniority requirements
- Predicted time-to-fill insights
- Kanban-style candidate pipeline
- One-click stage progression with audit history
- AI-powered candidate matching and fit analysis

### 👥 Talent Pool
- Searchable candidate database
- Skill and location filters
- Candidate profile management
- Cross-role application tracking

### 🤖 AI Capabilities
- Resume parsing from unstructured text
- Predictive candidate matching
- AI-generated fit rationale
- Interview question generation with evaluation rubrics
- Deterministic fallback scoring

## Database Schema

Core entities:
- `jobs`
- `candidates`
- `applications`
- `match_scores`
- `application_stage_history`
- `ai_artifacts`

## Tech Stack

- Frontend: React + TypeScript
- Backend: Node.js
- Database: PostgreSQL
- AI: Claude Sonnet 4.5
- Styling: High-Fidelity SaaS Design System

## Roadmap

- Automated stage advancement
- Candidate email outreach
- Offer management
- Bulk actions
- Calendar integration
- Advanced hiring analytics

## Goal

Talisma enables recruiters to spend less time on administrative tasks and more time hiring the right talent through AI-powered recruitment intelligence.
